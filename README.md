# AgentOS

Your own Claude Code agent 24/7 — Telegram bot + tasks + routines + Mini App.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/try-agent-os/agentos/main/install.sh | bash -s -- --domain agent.example.com
```

See [releases](https://github.com/try-agent-os/agentos/releases) for release
notes, the bare-metal tarball, and the `stable.json` channel manifest.
